This is a simple, command-line Unit Converter program built in Python. It allows users to easily convert between common units of Length, Weight, and Temperature. It's a great example of a basic interactive Python application using functions and a main loop.
Features üìã
Length Conversion: Converts meters to centimeters, inches, and feet.
Weight Conversion: Converts kilograms to grams, pounds, and ounces.
Temperature Conversion: Converts Celsius to Fahrenheit and Kelvin.
Interactive Menu: A user-friendly, loop-based menu for selecting the desired conversion type.

Running the Script
